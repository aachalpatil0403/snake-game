#Snake Game 🐍

A classic Snake Game built using Python and Pygame. The player controls a snake that grows longer each time it eats food, while avoiding collisions with walls and itself.

#Features
Classic snake gameplay.
Smooth movement and increasing snake length.
Score display during gameplay.
Game over screen with the option to restart or quit.
Simple and colorful graphics.

#Installation
Make sure you have Python installed (version 3.7+ recommended).
Install Pygame if you don’t already have it:
pip install pygame

Run the game:
python snake_game.py

How to Play
*Arrow keys to move the snake:
*UP → Move up
*DOWN → Move down
*LEFT → Move left
*RIGHT → Move right
*Eat the green squares to grow your snake.

Avoid hitting the walls or your own tail.
When the game ends, press:
~C to play again
~Q to quit
